Blackjack WPF Game
------------------

This project contains a WPF-based Blackjack game with basic UI, animations,
and client-server architecture support.

Files included:
- MainWindow.xaml.cs: UI logic and animations for the WPF interface.
- BlackjackGame.cs: Game logic for deck, player, dealer, and gameplay.
- Client.cs & Server.cs: TCP-based networking for multiplayer support.

To run:
1. Open in Visual Studio (WPF App template).
2. Add card images to /Images in your project.
3. Run the WPF project and click Hit/Stand.

You can expand on this by adding more animations, betting logic, and multiplayer UI.
